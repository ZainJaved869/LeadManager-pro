1. Overview

LeadManager Pro is a modern, multi‑tenant CRM built with Laravel 12. It helps sales teams manage leads, pipelines, customers, tasks, and communications – all powered by AI (Groq) and backed by a full subscription system.

  • ✅ Multi‑Tenant Architecture
  • ✅ Lead Management (CRUD + Import/Export + Notes + Attachments)
  • ✅ Kanban Pipeline (drag‑and‑drop)
  • ✅ AI Integration (Groq) – Email / Follow‑up / Proposal / Summary
  • ✅ WhatsApp Integration
  • ✅ Email Marketing
  • ✅ Task Management + Reminders + Calendar
  • ✅ Customer Management (Companies + Contacts + Communication History)
  • ✅ Analytics Dashboard (Chart.js)
  • ✅ Subscription System (Free, Basic, Pro, Enterprise)
  • ✅ Admin Panel

2. System Requirements

  • PHP: 8.1 or higher (8.2+ recommended)
  • Laravel: 12.x
  • Database: MySQL 5.7+ / MariaDB 10.4+
  • Extensions: BCMath, Ctype, Fileinfo, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML, cURL
  • Memory: 256MB+
  • Storage: 100MB+

3. Installation Guide

Step 1: Upload Files

Upload all package files to your hosting account using cPanel File Manager or FTP.

Step 2: Create Database

Create a new MySQL database and database user.

Step 3: Configure Environment File

Rename .env.example to .env.

Update database credentials:

DB_DATABASE=your_database
DB_USERNAME=your_username
DB_PASSWORD=your_password

Update application URL:

APP_URL=https://yourdomain.com

Step 4: Install Dependencies

composer install --optimize-autoloader --no-dev

Step 5: Generate Application Key

php artisan key:generate

Step 6: Run Database Migration

php artisan migrate --seed
💡 Note: If you encounter foreign key errors, run the complete SQL schema (provided in the package) first, then run php artisan migrate --force to mark migrations as complete.

Step 7: Create Storage Link

php artisan storage:link

Step 8: Optimize Application

php artisan config:cache
php artisan route:cache
php artisan view:cache

Step 9: Configure Permissions

chmod -R 775 storage
chmod -R 775 bootstrap/cache

Step 10: Configure Cron Job

* * * * * php /path-to-project/artisan schedule:run >> /dev/null 2>&1

Step 11: Access Application

User Login: https://yourdomain.com/login

Admin Login: https://yourdomain.com/admin/login

🔑 Default Admin Credentials:
Email: admin@leadmanager.com
Password: password

Installation Complete! Your CRM SaaS application is now ready to use.

8. API Integrations

Groq (AI)

  • Requires GROQ_API_KEY in .env
  • Supported models: llama3-8b-8192, llama3-70b-8192, mixtral-8x7b-32768

WhatsApp Cloud API

  • Requires WHATSAPP_ACCESS_TOKEN and WHATSAPP_PHONE_NUMBER_ID
  • Webhook URL: https://yourdomain.com/api/whatsapp/webhook
  • Subscribe to messages and message_deliveries events

9. Troubleshooting

  • 500 Internal Server Error: Check storage/logs/laravel.log. Run composer dump-autoload and php artisan config:clear.
  • Table not found: Run php artisan migrate or re‑import the full SQL schema.
  • AI not working: Verify GROQ_API_KEY is correct and clear config cache.
  • Notifications not showing: Run php artisan reminders:send manually to test.
  • 404 on Export/Import: Ensure custom routes are placed before Route::resource('leads').
  • Logo not showing: Make sure logo.png exists in public/ and permissions are correct.
